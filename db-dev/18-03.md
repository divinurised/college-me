## Item 4

```sql
SET serveroutput ON
DECLARE
  V_CHAR VARCHAR2(16) := '42 is the answer';
  V_NUM NUMBER := TO_NUMBER(SUBSTR(V_CHAR, 1, 2);
BEGIN
dbms_output.put_line(V_CHAR)
dbms_output.put_line(V_NUM)
END;
```
